# 愛我家鄉-社區營造大作戰

這是以教室觸碰大螢幕為主要使用情境的四人社區營造攻防卡牌遊戲。

## 主要入口

- `index.html`：老師開局入口，可設定玩家數、真人玩家與 AI 代玩。
- `player_window/player-window.html`：玩家觸碰桌主畫面。

## 主要資料夾

- `assets/`：桌面底圖、背景音樂與共用素材。
- `card_bank/`：卡牌圖資與分類資料。
- `doc/`：Excel 開發指令工作簿。
- `outputs/`：仍保留的玩家視窗設計輸出與預覽。
- `player_window/`：玩家版遊戲視窗。
- `workbook/`：開發備份、備份索引與回復說明。

## 目前遊戲方向

- 不再保留主控戰情室 demo。
- 遊戲直接從 `index.html` 進入觸碰大螢幕玩家版。
- 遊戲結束條件包含 30 分鐘時間結束或玩家倒閉。
- 結局會顯示玩家分析星芒圖，對應 108 課綱社會科學習表現。

## 測試

```powershell
node smoke-test.js
```

此測試會確認卡池、首頁入口與玩家視窗必要結構。
