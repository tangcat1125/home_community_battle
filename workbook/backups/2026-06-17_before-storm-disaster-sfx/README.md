## 備份目的

在加入烏雲消散、防禦清雲、災害事件與音效加強前，保存可回復版本。

## 備份檔案

- `player-window.html`
- `build-command-workbook.mjs`
- `backup-index.md`

## 修改後重點

- 烏雲會依回合計數在兩回合後自動散去。
- 防禦卡成功觸發時，會直接清除被攻擊玩家身上的烏雲。
- 攻擊、防禦成功、烏雲密佈下雨聲與災害音效加大。
- 新增隨機地震與颱風事件。
- 地震或颱風會掀掉所有玩家桌牌，包含攻擊、防禦、陷阱、加成與地方主題。
- 被掀掉的桌牌會進入墳場。
- 災害後玩家遊客與基礎人潮減半。
